# 简介
腾讯云mall开发者工具sdk, 用于调用云mall的openapi接口, 使用前需要申请密钥对鉴权

# 依赖环境
1. jdk7版本及以上
2. 申请SecretID、SecretKey

# 依赖引用
```
        <dependency>
            <groupId>com.tencent.sr.rmall</groupId>
            <artifactId>rmall-api-sdk</artifactId>
            <version>1.0.0.RELEASE</version>
        </dependency>
```
